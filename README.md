# 💧 Aquafine – Premium Water Delivery Platform

A modern full-stack e-commerce platform for bottled water delivery, designed to provide a seamless shopping experience with secure payments, persistent user sessions, real-time order management, and a dedicated admin dashboard.

Built using **React Native (Expo)** for the customer mobile application and **React.js** for the admin dashboard, Aquafine delivers a production-ready shopping experience similar to leading e-commerce platforms.
